# Systeme_Detection_de_Vol-Shoplifting
Ce projet consiste en un système intelligent capable de détecter des comportements suspects et des actes de vol en temps réel à partir de flux vidéo. Il repose sur deux modèles d’intelligence artificielle :  Détection de personnes Détection de comportements de vol  L’objectif est d’automatiser la surveillance et d’assister les systèmes de sécurité.
